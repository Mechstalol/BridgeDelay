Command Reference

Control BridgeDelay through simple texts.

STATUS

Get the current bridge status and travel time.

THRESHOLD {minutes}

Set how many minutes of delay should trigger an alert.

WINDOW {start}-{end}

Specify quiet hours when you don't want alerts.

STOP

Pause all alerts until you send START.

START

Resume alerts after sending STOP.
