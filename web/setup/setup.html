Set your thresholds

To make sure you only get alerts for significant delays, set northbound and southbound thresholds in user settings or through text by sending "Threshold NB {time}" or "Threshold SB {time}". By default both values are 20 minutes.

Alerts repeat only if the delay changes by 15 minutes or the severity level shifts since the last message.
