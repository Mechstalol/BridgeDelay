Setup Guide

Get started with BridgeDelay in minutes.

Set your notification windows

BY DEFAULT YOU WILL GET NOTIFICATIONS FROM 7AM - 9AM AND 5PM-7PM. You can adjust this at any time in user settings or through SMS.

Set your threshold

BY DEFAULT YOU WILL ONLY GET AN SMS IF THE DELAY IS 20 MINUTES MORE THAN NORMAL. You can adjust this at any time in user settings or through text by sending "Threshold {time in minutes}". By default this value is 20 minutes.

Get familiar with the commands

There are multiple commands that will make your experience with my app smoother and more efficient. To see a list of commands, click here.
